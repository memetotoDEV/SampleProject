このサイトについて

Spring BootとThymeleafを使用して作成されたシンプルなWebアプリケーションです。

使用技術

  • Spring Boot 3.2.0
  • Spring Web MVC
  • Thymeleaf Template Engine
  • Bootstrap 5
  • Maven
  • Java 17

特徴

  • レスポンシブデザイン
  • モダンなUI/UX
  • 高速な起動時間
  • 簡単な拡張性
  • クリーンなコード構造
  • 日本語対応
